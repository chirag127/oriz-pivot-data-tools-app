# Oriz Pivot — Data + Spreadsheet Tools

Data and spreadsheet utilities under the **Pivot** product brand — named after the spreadsheet pivot table, the workhorse of data analysis.

## Stack

- [Astro 6](https://astro.build/) — static-first site framework
- [React 19](https://react.dev/) — interactive islands
- [Tailwind CSS v4](https://tailwindcss.com/) — utility-first styling
