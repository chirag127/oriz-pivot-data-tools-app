# Oriz Pivot — Data tools

> Data and spreadsheet utilities under the Pivot product brand — named after the spreadsheet pivot table, the workhorse of data analysis.

**Live at**: <https://data.oriz.in> · **Status**: scaffold

## What this is

Browser-based data and spreadsheet utilities: CSV/JSON/TSV transforms, pivot operations, sorting, filtering, deduplication, and the rest of the spreadsheet workbench.
